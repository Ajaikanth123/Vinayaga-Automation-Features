Enterprise Feature Overview

17 powerful modules built for diagnostic centers.

1
assignment

Patient Referral Form System

Digital form for doctors to submit scan requests with patient details, scan type, and tooth selection.

2
badge

Patient ID Generation

Automatic unique patient ID generation during referral submission.

3
cloud_upload

Scan Upload System

Secure upload interface for DICOM scan files mapped with patient ID and scan details.

4
cloud

Cloud Scan Storage

Secure storage system for large DICOM scan files using centralized cloud infrastructure.

5
preview

Web-Based DICOM Viewer

Browser-based DICOM viewer with zoom, pan, slice navigation, and scan visualization.

6
share

Automated Scan Sharing

Automatic generation of secure viewing links once the scan upload is completed.

7
chat

WhatsApp Notification Automation

Automated notification system that sends scan-ready alerts to doctors through WhatsApp.

8
mail

Email Notification System

Automated email alerts containing secure scan viewing links.

9
dashboard

Admin Dashboard

Centralized dashboard for managing users, scan uploads, patient records, and system activity.

10
account_tree

Multi-Branch Management

Support for managing multiple diagnostic center branches within one centralized platform.

11
admin_panel_settings

Role-Based Access Control

Different system roles for administrators, staff members, and doctors with controlled permissions.

12
link

Secure Scan Access Links

Encrypted and time-limited scan viewing links for doctors.

13
troubleshoot

Activity Logs

System logs that record user actions such as scan access, uploads, and login activity.

14
manage_search

Search & Patient Lookup

Advanced search system to find scans using patient name, ID, or scan date.

15
smartphone

Mobile-Friendly Viewer

Fully responsive viewer allowing doctors to access scans through mobile devices or tablets.

16
enhanced_encryption

Data Encryption

Encryption system for protecting stored medical scan data and patient information.

17
web

Dedicated Clinic Website Service

Development of a dedicated website for the clinic integrated with the DICOM CONNECT platform.
